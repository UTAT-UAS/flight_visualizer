# Flight Visualizer


## Developing

```sh
pnpm config set store-dir ~/.local/share/pnpm/store # if in the container
pnpm install
pnpm run dev
# or start the server and open the app in a new browser tab
pnpm run dev --open
```

## Building

```sh
pnpm run build
```

You can preview the production build with `npm run preview`.
